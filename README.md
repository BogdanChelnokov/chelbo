## 📋 О проекте

**Chelbo** — это современный российский кроссплатформенный мессенджер с открытой архитектурой. Он работает прямо в браузере и как PWA без необходимости установки приложения. Проект разработан в качестве дипломной работы в Академии ТОР.

> **Демо:** [https://chelbo.ru](https://chelbo.ru)

## ✨ Особенности

### Основные функции

| Функция | Описание |
|---------|----------|
| 💬 **Обмен сообщениями** | Мгновенная доставка через WebSocket |
| 👥 **Личные и групповые чаты** | Диалоги 1-на-1 и групповые обсуждения |
| 📎 **Обмен файлами** | Изображения (до 10 MB) и документы (до 50 MB) |
| 🎙️ **Голосовые сообщения** | Запись и отправка голоса в чате |
| 📹 **Видеосообщения** | Запись и отправка коротких видео |
| ↩️ **Ответы на сообщения** | Цитирование и ответ (как в Telegram) |
| 📤 **Пересылка сообщений** | Пересылка в другие чаты |
| ✓ **Статусы прочтения** | Доставлено / прочитано |
| ⌨️ **Индикатор печати** | Отображение когда собеседник печатает |
| 🔴 **Счётчик непрочитанных** | Бейдж с количеством новых сообщений |

> **Важно:** Отправка медиафайлов (изображения, документы, голосовые и видеосообщения) в текущей версии доступна только на ПК. На мобильных устройствах эта функция будет добавлена в следующих обновлениях.

### Расширенные возможности

| Функция | Описание |
|---------|----------|
| 📱 **PWA поддержка** | Установка на телефон без App Store |
| 💻 **Адаптивный дизайн** | Работает на ПК, планшетах и телефонах |
| 🌙 **Тёмная тема** | Тёмная цветовая схема с красным акцентом |
| 🔍 **Поиск сообщений** | Полнотекстовый поиск по сообщениям |
| 🟢 **Статус онлайн** | Отображение когда пользователь в сети |

## 🏗 Архитектура (продолжение)

### Компоненты системы

#### 1. Клиентская часть (Frontend)

| Компонент | Описание | Технологии |
|-----------|----------|------------|
| **Компоненты** | UI элементы страницы (список чатов, окно сообщений, форма входа) | Vue 3 Composition API |
| **Хранилище состояния** | Управление данными пользователя, чатов, сообщений | Pinia |
| **WebSocket клиент** | Поддержание постоянного соединения с сервером, обработка входящих сообщений | Native WebSocket API |
| **HTTP клиент** | Отправка запросов на REST API, загрузка файлов | Axios |
| **Роутинг** | Навигация между страницами (логин, регистрация, чат, настройки) | Vue Router |
| **Медиа рекордер** | Запись голосовых и видеосообщений | MediaRecorder API |

#### 2. Серверная часть (Backend)

| Компонент | Описание | Технологии |
|-----------|----------|------------|
| **HTTP сервер** | Обработка REST запросов, маршрутизация | Gorilla Mux |
| **WebSocket хаб** | Управление подключениями, рассылка сообщений | Gorilla WebSocket |
| **Модуль аутентификации** | Регистрация, вход, JWT токены | JWT, bcrypt |
| **Модуль чатов** | Создание чатов, управление участниками | Go, MySQL |
| **Модуль сообщений** | Отправка, получение, удаление, статусы | Go, MySQL |
| **Модуль файлов** | Загрузка, хранение, выдача файлов | Go, local storage |
| **Модуль AI** | Обработка запросов к AI-ассистенту | Go, имитация нейросети |
| **Redis клиент** | Кэширование, Pub/Sub для горизонтального масштабирования | go-redis |

#### 3. База данных

| Таблица | Поля | Описание |
|---------|------|-----------|
| **users** | id, email, password_hash, name, avatar_url, bio, is_active, is_verified, last_seen, created_at, updated_at | Информация о пользователях |
| **chats** | id, type (private/group), title, avatar_url, created_by, created_at, updated_at | Информация о чатах |
| **chat_participants** | chat_id, user_id, role, joined_at, last_read_at | Участники чатов |
| **messages** | id, chat_id, sender_id, text, file_url, file_type, reply_to_id, is_deleted, created_at | Сообщения |
| **message_statuses** | message_id, user_id, is_delivered, is_read, delivered_at, read_at | Статусы доставки/прочтения |
| **files** | id, user_id, filename, original_name, file_path, file_size, mime_type, created_at | Метаданные загруженных файлов |
